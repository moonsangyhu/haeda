## Flow 1: 회원가입 / 로그인

```
┌─────────────┐     ┌──────────────┐     ┌──────────────────┐     ┌───────────┐
│  스플래시    │────▶│  로그인 화면  │────▶│  카카오 OAuth     │────▶│ 프로필    │
│  스크린      │     │              │     │  인증 웹뷰        │     │ 설정 화면  │
└─────────────┘     │ [카카오 로그인]│     │                  │     │           │
                    └──────────────┘     └──────────────────┘     │ - 닉네임   │
                                                                   │ - 프로필   │
                                                                   │   사진     │
                                                                   │ [완료]     │
                                                                   └─────┬─────┘
                                                                         │
                                                                         ▼
                                                                   ┌───────────┐
                                                                   │  내 페이지 │
                                                                   └───────────┘
```

- **최초 가입 시**: 카카오 인증 → 프로필 설정 → 내 페이지
- **재방문 시**: 자동 로그인 → 내 페이지 (토큰 유효 시)

## Flow 2: 내 페이지

```
┌──────────────────────────────────────────────┐
│                 내 페이지                      │
│                                              │
│  ┌──────────────────────────────────────┐    │
│  │ 프로필 영역 (사진, 닉네임)              │    │
│  └──────────────────────────────────────┘    │
│                                              │
│  [챌린지 만들기]                              │
│                                              │
│  ── 참여 중인 챌린지 ──                       │
│  ┌─────────────────────────┐                 │
│  │ 🏃 운동 30일            │ ──tap──▶ 챌린지 │
│  │ 달성률 73% · 참여자 5명  │         공간    │
│  └─────────────────────────┘                 │
│  ┌─────────────────────────┐                 │
│  │ 📖 독서 습관             │ ──tap──▶ 챌린지 │
│  │ 달성률 40% · 참여자 3명  │         공간    │
│  └─────────────────────────┘                 │
│                                              │
│  ── 완료된 챌린지 ──                          │
│  ┌─────────────────────────┐                 │
│  │ 🏅 물 마시기 (완료)      │                 │
│  │ 달성률 90% · 배지 획득   │                 │
│  └─────────────────────────┘                 │
│                                              │
│  ─────────────────────────                   │
│  [내 페이지]  [탐색]  [알림]                  │
└──────────────────────────────────────────────┘
```

- 참여 중 / 완료 챌린지를 구분하여 표시
- 각 카드에 카테고리, 제목, 달성률, 참여자 수 요약
- 하단 탭: 내 페이지 | 탐색 | 알림

```

┌──────────────────────────┐

│  - 설명                    │

└────────────┬─────────────┘
             ▼
┌──────────────────────────┐

│  - 시작일 ~ 종료일         │

│  [생성하기]                 │

             ▼
┌──────────────────────────┐
│  생성 완료!                │
│                           │
│  초대 코드: ABCD1234       │

│                           │
│  [챌린지로 이동]            │
└──────────────────────────┘
```

## Flow 5: 챌린지 공간 (달력 뷰) — 핵심 화면

```
┌──────────────────────────────────────────────┐
│  ◀  운동 30일 챌린지  (참여자 5명)            │
│                                              │
│  ──────── 2026년 4월 ────────                │
│  일  월  화  수  목  금  토                    │
│            1    2    3    4                    │
│       ┌──┐┌──┐                               │
│       │🌸││🌸│  ← 전원 인증 = 계절 아이콘     │
│       └──┘└──┘                               │
│   5    6    7    8    9   10   11             │
│       ┌──┐┌──┐┌──┐                           │
│       │🌸││😀││😀│  ← 일부만 인증 = 썸네일    │
│       └──┘│😀│└──┘                           │
│            └──┘                               │
│                          ...                  │
│                                              │
│  ──────── 오늘 (4월 4일) ────────             │
│  아직 인증하지 않았어요!                        │
│  [인증하기]                                    │
│                                              │
│  ─────────────────────────                   │
│  [내 페이지]  [탐색]  [알림]                  │
└──────────────────────────────────────────────┘
```

### 달력 아이콘 규칙
| 상태 | 표시 |
|------|------|
| 아무도 인증 안 함 | 빈 칸 |
| 일부 인증 | 인증한 사람들의 프로필 사진 썸네일 (작게 배열) |
| **전원 인증** | **계절 아이콘으로 변환** (봄🌸 여름🌿 가을🍁 겨울❄️) |

### 계절 판정 기준
| 계절 | 기간 | 아이콘 |
|------|------|--------|
| 봄 | 3월 ~ 5월 | 꽃 (🌸) |
| 여름 | 6월 ~ 8월 | 나뭇잎 (🌿) |
| 가을 | 9월 ~ 11월 | 단풍 (🍁) |
| 겨울 | 12월 ~ 2월 | 눈꽃 (❄️) |

## 화면 구조 요약

```
앱
├── 로그인 (카카오 OAuth)
├── 프로필 설정 (최초 1회)
├── 메인 탭
│   ├── 내 페이지
│   │   ├── 참여 중 챌린지 목록
│   │   └── 완료된 챌린지 목록
│   ├── 탐색
│   │   └── 공개 챌린지 목록
│   └── 알림
│       └── 푸시 알림 목록
├── 챌린지 생성 (Step 1~2 + 완료)
├── 챌린지 공간
│   ├── 달력 뷰 (월별)
│   ├── 날짜별 인증 현황
│   ├── 인증 상세 (사진+일기+댓글)
│   └── 인증 작성
└── 챌린지 완료 결과
```
